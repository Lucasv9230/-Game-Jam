Règles du jeu

Bienvenue dans le jeu ! Voici les règles :

  • Atteindre l’hôpital avec le temps impartie pour sauver le patient.
  • Éviter les obstacles sur la route pour ne pas réduire les points de vie du patient.
  • Gérer la vitesse de l’ambulance : accélérer pour gagner du temps, ralentir pour éviter les collisions.
  • Choisir un mode de difficulté : Easy, Medium ou Hard.
  • Si la barre de vie du patient tombe à zéro, c’est Game Over.
